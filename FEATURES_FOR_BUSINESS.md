# TOFA CRM - Features Overview
## A Simple Guide for Non-Technical Users

## What is TOFA CRM?

TOFA CRM is a smart system designed to help your academy manage leads (potential students) from the moment they express interest until they become members. It tracks every interaction, helps prioritize your work, and gives you insights into how your business is performing.

## üìÖ Planning & Organization

### Command Center (Your Daily Hub)

**What it is:**
A unified hub that shows different information based on your role.

**For Sales Team (Team Leads & Regular Users):**
- **Today's Progress**: How many leads you've updated today vs. how many are due
- **Unscheduled**: Leads without a follow-up date set
- **Overdue**: Leads that should have been contacted already
- **Trial Show-Up Rate**: How many trials showed up today
- **Action Queue**: List of leads that need your attention, sorted by priority

**For Coaches:**
- **Session Coverage**: How many sessions you've completed today
- **New Arrivals**: Students scheduled for trial today in your batches
- **Skill Report Backlog**: Members who need skill reports (haven't had one in 60+ days)
- **Capacity Warnings**: Batches that are getting full (>90% capacity)
- **Action Queue**: Your batches and trial arrivals for today
- **Quick Access**: Click a batch to go directly to check-in

**Why it matters:**
- See everything you need in one place
- Know exactly what to focus on
- Different views for different roles
- Start your day with clear priorities

### Daily Task Queue

**What it is:**
Your personalized to-do list for today, automatically prioritized.

**How it works:**
- Shows you exactly what needs attention today
- Prioritizes overdue items first
- Then shows today's tasks
- Finally shows upcoming tasks

**Features:**
- See how many calls you have today
- See how many are high-priority
- Progress tracking (how many tasks completed)
- **Use it to**: Start each day knowing exactly what to do

### Calendar View

**Month Overview:**
- See your entire month at a glance
- Color-coded to show busy days (red) vs. light days (green)
- See how many trials are scheduled each day
- See how many follow-ups are due each day
- Click any day to see details

**Planner Pane** (on Leads page):
- Mini calendar showing the month
- Click any day to see leads scheduled for that day
- Quick view of who needs attention
- Filter your leads table by date
- **Use it to**: Plan your week and balance your workload

### Trial Check-In (Mobile-Friendly)

**What it is:**
A simple page optimized for coaches using phones/tablets on the field.

**Features:**
- **Batch Selection**: Choose which batch you're marking attendance for
- Shows all students in the selected batch (both trial and permanent members)
- Large, easy-to-tap buttons (green for "Present", red for "Absent")
- Quick search by player name
- Summary showing batch name, time, and student count
- "Complete Session" button to finish and return to Command Center
- Navigate from Command Center directly to check-in for a specific batch

**Use it to:**
- Mark attendance quickly during sessions
- No need to remember who showed up
- Update status immediately
- Track attendance for all your batches

## üë• Team Management

### User Roles

**Team Lead:**
- Can see all leads from all centers
- Can manage users (create Team Leads, Regular Users, Coaches, Observers)
- Can assign leads to centers
- Can create and manage training batches
- Can assign coaches to batches
- Has access to all features and analytics

**Regular User:**
- Sees only leads from their assigned centers
- Can update leads and add notes
- Focused view of their work
- Sales-focused features

**Coach:**
- Sees only leads from their assigned training batches
- Cannot see contact information (phone, email) - kept private for data security
- Can mark attendance for their batches
- Can create skill reports for members
- Mobile-friendly check-in interface
- Field-focused features

**Why it matters:**
- Everyone sees what they need
- No confusion about responsibilities
- Clear ownership of leads
- Data privacy for parents (coaches don't see contact info)
- Role-appropriate features and views

### Center Management

**What centers are:**
Your different academy locations or programs.

**Features:**
- Create new centers
- Assign leads to specific centers
- Assign users (including coaches) to centers
- Track performance by center
- Filter leads by center
- **Use it to**: Organize leads by location and see which centers perform best

### Batch Management (Training Groups)

**What batches are:**
Training groups or sessions (e.g., "U9 Morning Batch", "U11 Evening Batch")

**Features:**
- Create training batches with name, age group, and schedule
- Assign multiple coaches to each batch
- Set maximum capacity (prevents overbooking)
- Choose which days of the week the batch runs (Mon-Sun)
- Set start and end times
- See all coaches assigned to each batch at a glance
- Edit coach assignments anytime

**Why it matters:**
- Organize students into training groups
- Know who's responsible for each group
- Prevent scheduling conflicts
- Track capacity to avoid overcrowding
- Flexible scheduling (morning, evening, weekend batches)

**Use it to:**
- Create structured training programs
- Assign coaches to their groups
- Plan schedules efficiently
- Track which coaches handle which groups

### Coach Management

**Creating Coaches:**
- When creating a coach account, you must assign them to at least one center
- Coaches can be assigned to multiple centers if needed
- Coaches are automatically linked to batches (see below)

**Assigning Coaches to Batches:**
- Each batch must have at least one coach assigned
- You can assign multiple coaches to the same batch (team coaching)
- When creating a batch, you select which coaches will handle it
- You can change coach assignments later if needed

**Coach Privacy:**
- Coaches cannot see parent contact information (phone, email, address)
- This protects parent privacy while still allowing coaches to work with students
- Coaches see student names, age groups, and batch information
- Only sales team can see full contact details

**Why it matters:**
- Protects parent privacy
- Clear responsibility for each batch
- Supports team coaching (multiple coaches per batch)
- Coaches focus on field work, not sales

### Attendance Tracking

**Mobile Check-In System:**
- Coaches use their phone/tablet to mark attendance
- Simple, one-tap interface: "Present" or "Absent"
- Shows all students in the selected batch
- Quick search to find students by name
- Works offline-friendly (optimized for field use)

**Features:**
- Select which batch you're marking attendance for
- See all students (both trial and permanent members) in that batch
- Large buttons for easy tapping (no mistakes in the sun)
- Search bar to quickly find a student
- Summary showing total students in the batch
- "Complete Session" button when done

**Automatic Updates:**
- When you mark someone as "Present", the system automatically:
  - Updates their status to "Trial Attended" (if they were on trial)
  - Adds a note about when they attended
  - Records who marked the attendance (the coach)

**Why it matters:**
- Track who shows up to sessions
- Identify attendance patterns
- No need to remember and write down later
- Instant updates in the system
- Parents can see attendance history (future feature)

**Use it to:**
- Mark attendance during sessions (takes seconds)
- Track student participation
- Identify students who need follow-up
- Generate attendance reports

## üéÅ Smart Features

### Referral System

**What happens:**
When a lead becomes a member, you're automatically prompted to ask for referrals.

**Features:**
- Pre-written WhatsApp message
- Personalized with player's name
- Includes a referral link for tracking
- One click to send
- Or skip if not the right time

**Use it to:**
- Grow through word-of-mouth
- Reward happy members
- Track which members bring referrals

### Age Group Migration

**What it does:**
Automatically calculates a child's correct age group based on their date of birth.

**Features:**
- Alerts you when a player needs to move up an age group
- One-click confirmation to update
- Prevents players from being in wrong age groups

**Use it to:**
- Keep age groups accurate
- Move players up automatically as they age
- Ensure fair competition

### Skill Reports

**What it is:**
A way for coaches to track player development over time.

**Features:**
- Rate players on Technical Skill, Fitness, Teamwork, and Discipline (1-5 stars each)
- Add coach's notes
- View previous reports to see progress
- Only available for members (Joined status)

**Use it to:**
- Track player improvement
- Share progress with parents
- Identify areas for development

### Duplicate Prevention

**What it does:**
Prevents the same lead from being entered multiple times.

**How it works:**
- Checks name, phone, and email combination
- If duplicate found, updates existing lead instead of creating new one
- Refreshes the lead (marks as "New" if it was "Dead")
- Adds a note about where the duplicate came from

**Use it to:**
- Keep your database clean
- Avoid wasting time on duplicates
- See all interactions with the same person

## üéØ Real-World Scenarios

### Scenario 1: Starting Your Day
1. Open the Dashboard - see key metrics
2. Check Daily Task Queue - see your prioritized list
3. Start with red (rotting) leads first
4. Work through today's tasks
5. Check progress bar - celebrate when you hit 100%!

### Scenario 2: New Lead Comes In
1. Lead appears automatically (from Meta ad or import)
2. System scores the lead (you see the stars)
3. You see it's "New" and needs first contact
4. Click the lead, update status to "Called", add notes
5. Set follow-up date for tomorrow
6. Lead now shows as "Green" (fresh)

### Scenario 3: Following Up
1. Open Planner Pane, click today's date
2. See list of leads scheduled for today
3. Click a lead, see their history in Activity Feed
4. Call them, update status, add notes
5. If they book a trial, status changes to "Trial Scheduled"
6. System automatically creates reminder tasks

### Scenario 4: Trial Day (Coach View)
1. Open Command Center on your phone
2. See "New Arrivals" count for today
3. Click on a batch card in Action Queue
4. Automatically navigates to Check-In page for that batch
5. See all students in the batch
6. As players arrive, tap "Present" (green button)
7. Status automatically updates, attendance recorded
8. Tap "Complete Session" when done
9. Return to Command Center, see updated metrics

### Scenario 5: Creating a New Batch (Team Lead)
1. Go to Batches page
2. Click "Add New Batch"
3. Enter batch name (e.g., "U9 Morning Batch")
4. Select center and age category
5. Choose which days it runs (e.g., Mon, Wed, Fri)
6. Set start and end times
7. Set maximum capacity (e.g., 20 students)
8. **Select coaches** - choose one or more coaches from the list
9. Click "Create Batch"
10. Batch appears in table with coaches listed
11. Can edit coach assignments anytime using "Assign Coach" button

### Scenario 6: Member Joins
1. Update lead status to "Joined"
2. Referral modal appears automatically
3. Click "Send Referral Request"
4. WhatsApp opens with pre-filled message
5. Send to parent, track referrals via link

### Scenario 7: Coach's Day (Field Operations)
1. Open Command Center - see session coverage and new arrivals
2. Click on "Morning U9 Batch" in Action Queue
3. Navigate to Check-In page for that batch
4. See 15 students listed
5. As students arrive, tap "Present"
6. One student is absent - tap "Absent"
7. Complete session, return to Command Center
8. See updated "Session Coverage" metric (e.g., 2/3 sessions completed)
9. Check "Skill Report Backlog" - see which members need reports
10. Create skill report for a member (rate skills, add notes)

### Scenario 8: End of Day Review
1. Check Dashboard - see today's metrics
2. Review conversion rates - see how you're performing
3. Check at-risk members - see who needs attention
4. Review abandoned leads - see what you missed
5. Plan tomorrow using calendar view
6. **For Coaches**: Review attendance records, check capacity warnings

## üéØ Key Differences: Sales Team vs. Coaches

### What Sales Team Sees:
- All contact information (phone, email, address)
- All leads from their assigned centers
- Can update lead status and sales information
- Focus on conversion and follow-up

- **Privacy**: Parent contact information stays with sales team

- **Clarity**: No confusion about responsibilities
- **Security**: Data access is restricted appropriately
- **Efficiency**: Each role gets features optimized for their work
