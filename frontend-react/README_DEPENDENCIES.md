# Frontend Dependencies

The frontend uses **Node.js/Next.js**, not Python. Dependencies are managed via `package.json`.

## Installation

```bash
cd frontend-react
npm install
```

## Dependencies

All frontend dependencies are listed in `package.json`. Key packages:

- **Next.js 14** - React framework
- **React 18** - UI library
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **React Query** - Data fetching
- **Axios** - HTTP client
- **Recharts** - Charts/graphs

## Backend Dependencies

Backend dependencies are in the root `requirements.txt` file (Python).

## Note

The frontend does **NOT** use Python or Streamlit. It's a pure React/Next.js application.
